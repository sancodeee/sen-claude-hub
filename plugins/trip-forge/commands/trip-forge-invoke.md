你现在必须立即调用已安装的 `trip-forge` skill 处理用户当前的旅行规划需求。

用户需求：
$ARGUMENTS

严格执行 skill 中唯一且完整的旅行攻略模板：

1. 先检查目的地、天数、同行人、出发城市、出行时间、预算和旅行风格。
2. 缺失信息时一次性追问，不要自行假设或生成文件。
3. 信息完整后联网核实天气、票价、营业时间、预约、交通班次和季节性开放。
4. 生成一份完整、自包含、移动端优先的 HTML 文件报告。
5. 不得省略模板中的六个维度、天气调度、置信度、B 计划或 HTML 质量要求。
